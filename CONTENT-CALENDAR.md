## European Market Content Notes

Since Europe is the primary market, content should:
- Use British English spelling (colour, optimise, catalogue) — or produce dual versions
- Reference EU platforms: Zalando, ASOS, Otto.de, Bol.com alongside Amazon/Shopify
- Note European GDPR-friendly data handling for file transfers
- Target UK-specific queries: "clipping path service UK", "photo editing service London"

Consider separate `/blog/uk/` or hreflang-tagged variants if traffic justifies it (Phase 3+).
